# L’art Gallery – UI/UX Design

This repository contains the UI/UX design for **L’art Gallery**, a mobile application focused on exploring museums and artworks.

## Project Overview
- Designed using **Figma**
- Focused on clean, elegant museum-style UI
- Includes interactive artwork exploration and ticket management
